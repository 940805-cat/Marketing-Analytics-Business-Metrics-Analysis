# Marketing Analytics & Business Metrics Analysis

## Project Overview

This project analyzes user behavior, sales performance, and marketing investment data to understand customer acquisition, retention, and profitability.

The goal was to identify the most effective marketing channels and provide data-driven recommendations to optimize business strategy.

## Objectives

- Analyze user activity and purchase behavior.
- Measure marketing campaign performance.
- Identify profitable and inefficient acquisition channels.
- Evaluate customer value and retention trends.

## Key Metrics Analyzed

- CAC (Customer Acquisition Cost)
- ROMI (Return on Marketing Investment)
- LTV (Lifetime Value)
- AOV (Average Order Value)
- Conversion behavior
- Customer cohorts

## Process

1. Data Preparation
- Imported and cleaned datasets using Python.
- Prepared user visits, purchases, and marketing cost data for analysis.

2. Exploratory Data Analysis
- Analyzed user behavior trends.
- Evaluated purchase patterns and customer retention.
- Compared marketing channel performance.

3. Business Analysis
- Calculated business KPIs.
- Identified channels with better profitability.
- Generated recommendations for budget optimization.

## Tools & Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Data Visualization
- Cohort Analysis
- Business Analytics
- KPI Analysis

## Results

The analysis identified opportunities to improve marketing efficiency by reallocating investment toward channels with stronger profitability and better customer retention.

Recommendations were based on CAC, ROMI, LTV and user behavior metrics.
